
## 开发

```bash
# 克隆项目
git clone https://github.com/hsiangleev/element-plus-admin.git

# 进入项目目录
cd element-plus-admin

# 安装依赖
npm install

# 启动服务
npm run dev
```

浏览器访问 http://localhost:3002

## 发布

```bash
# 发布
npm run build

# 预览
npm run preview
```

## 其它

```bash
# eslint代码校验
npm run eslint

# stylelint代码校验
npm run stylelint
```
