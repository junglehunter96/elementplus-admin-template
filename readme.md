
## 开发

```bash

# 进入项目目录
cd cash-admin

# 安装依赖
yarn install

# 启动服务
yarn dev
```

浏览器访问 http://localhost:3002

## 发布

```bash
# 发布
yarn build

# 预览
yarn preview
```

## 其它

```bash
# eslint代码校验
yarn eslint

# stylelint代码校验
yarn stylelint
```
