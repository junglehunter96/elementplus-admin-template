
## 开发
### 参考
项目结构可参考如下项目和对应文档 基于这两项目改造
[vue-element-admin](https://github.com/PanJiaChen/vue-element-admin)

[element-plus-admin](https://github.com/hsiangleev/element-plus-admin)

```bash

# 进入项目目录
cd cash-admin

# 安装依赖
yarn install

# 启动服务
yarn dev
```

浏览器访问 http://localhost:3002
